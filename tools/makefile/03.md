## Makefile中与文件相关的函数

### dir与notdir函数

```makefile
file = ./files/a.c
fdir = $(dir $(file))
nfdir = $(notdir $(file))

main: demo.c
	echo $(fdir)
	echo $(nfdir)
```

dir函数主要书获取文件路径当中的目录部分，而notdir函数主要是获取文件路径当中文件名的部分，执行上面的makefile结果如下所示：
